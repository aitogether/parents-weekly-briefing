## 二、服务内容

「父母周报」是一款面向异地子女与年迈父母的家庭健康关怀工具，核心功能包括：

1. **健康数据记录**：父母端可记录每日步数、用药情况、情绪状态
2. **周报自动生成**：系统每周一自动生成上周健康简报，推送至子女端
3. **提醒与通知**：用药提醒、周报生成通知
4. **家庭绑定**：子女可通过邀请码绑定父母，建立家庭关系

**重要说明**：本小程序提供的是健康信息记录与提醒服务，**不是医疗诊断工具**，不提供医疗建议、处方或紧急救护。
