# 图标设计规范 · Icon Guidelines

> 核心原则：**安心、温和、记得住。** 宁可少一点、简单一点，也不要五颜六色。

## 1. App 主图标（Launcher Icon）

### 设计要求
- **极简图形**，二选一方案：
  - A）小房子轮廓 + 心形/心跳线
  - B）2 个简化人形（父母）+ 1 个小点（子女）在旁边
- **色板**：只用 2–3 种颜色
  - 主色：蓝绿 #2196F3 ~ #00BCD4
  - 暖色点缀：#FFB74D（暖橙，极少量）
- **风格**：扁平、线性、无复杂渐变
- **形状**：圆角方形（Android adaptive icon 标准），108×108dp
- **不在图标上放文字**

### 参考色板
| 用途 | 色值 |
|------|------|
| 主色 | #2196F3 → #00BCD4 |
| 暖色点缀 | #FFB74D（极少量） |
| 非活跃 | #9E9E9E |

## 2. 状态/功能图标（页面内）

### 基础要求
- **统一使用 Material Symbols Outlined** 一套为基础
- **一律线性风格**，不要实心卡通
- 线条粗细：1.5dp
- 颜色：尽量单色线条（深蓝灰 #455A64），只在状态灯上用彩色

### 固定映射
| 功能 | 图标 | 说明 |
|------|------|------|
| 步数 | 脚印/鞋子轮廓 | `directions_walk` |
| 用药 | 药丸/药瓶 | `medication` |
| 状态灯 | 简化圆点 | 绿/黄/红 |
| 回声 | 对话气泡 | `chat_bubble_outline` |

## 3. 数量控制

- **每个页面最多 3–4 个图标**
- 如果犹豫要不要加图标 → **先不加**

| 页面 | 允许的图标（最多） | 禁止的 |
|------|-----------|--------|
| 周报首屏 | 状态灯 + 用药 + 步数 | 额外装饰箭头、花边 |
| 用药页 | 药丸按钮 + 状态灯 + 回声气泡 | 复杂场景图 |
| App 首页 | 每个卡片 1 个小图标 | 每个按钮都加图标 |
| 设置页 | 齿轮 + 箭头 | 大量功能图标 |

## 4. 迭代优先级

1. **P0**：统一 App launcher 图标
2. **P1**：统一两屏首屏的图标（周报首屏 + App 首页）
3. **P2**：统一其他页面图标
4. **P3**：统一列表项小图标
