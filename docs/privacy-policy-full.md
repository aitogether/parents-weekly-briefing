# 家长周刊 - 隐私政策

**最后更新日期：2026年4月18日**

我们非常重视您的隐私保护。本隐私政策阐述了我们（"我们"、"平台"或"家长周刊"）如何收集、使用、存储、共享和保护您在使用"家长周刊"微信小程序及关联服务（以下简称"服务"）时的个人信息。

本政策依据《中华人民共和国个人信息保护法》(PIPL)、《通用数据保护条例》(GDPR)、《微信小程序平台规范》以及相关法律法规制定，涵盖了个人信息处理的8项核心要素：收集、使用、存储、共享、转移、公开、保护及您的权利。

---

## 1. 我们收集的个人信息类型

### 1.1 您主动提供的信息

**身份信息：**
- 微信昵称、头像：用于账户创建和个人化展示
- 家长/子女角色标识：用于区分用户类型和功能权限
- 邀请码：用于家庭成员的绑定和验证

**健康与用药信息：**
- 用药计划：药品名称、剂量、服用时间表
- 用药确认记录：确认日期、确认状态
- 健康简报：子女的每周健康状况摘要
- 焦虑量表评估结果：心理健康自我评估数据

**反馈与互动信息：**
- 对周报的反馈："子女回声"功能中的评价和意见
- 安全检查清单完成情况

### 1.2 系统自动收集的信息

**设备与日志信息：**
- 微信OpenID：微信用户唯一标识符（加密存储）
- 设备型号、操作系统版本、微信版本
- 服务日志：操作时间、功能使用记录、错误信息
- IP地址、访问时间、页面浏览记录

**微信运动数据（需您授权）：**
- 每日步数信息：用于生成家长活动报告
- 数据日期：对应的时间戳

**推送通知相关信息（需您授权）：**
- 订阅消息授权状态
- 提醒设置：用药提醒时间配置

## 2. 我们如何使用您的个人信息

我们依据合法、正当、必要的原则，将您的个人信息用于以下目的：

### 2.1 核心服务功能

**账户管理与身份验证：**
- 创建和管理您的账户
- 验证您的身份和权限
- 维护您与其他家庭成员的绑定关系

**健康管理与用药提醒：**
- 存储和管理用药计划
- 发送用药提醒通知
- 记录用药确认情况
- 生成每周用药依从性报告

**家庭协作：**
- 连接家长与子女账户，实现健康数据共享
- 允许子女向家长提供周报反馈
- 支持家长查看子女的用药情况

**数据分析与报告：**
- 生成每周健康简报
- 分析用药依从性趋势
- 提供焦虑量表历史记录
- 统计微信运动数据，鼓励健康生活方式

### 2.2 服务改进与安全

**服务优化：**
- 分析功能使用情况，改进用户体验
- 修复程序错误，提升系统稳定性
- 进行内部审计和数据安全监测

**安全防护：**
- 检测和防范欺诈、滥用行为
- 保护系统免受网络攻击
- 验证用户操作的真实性

## 3. 我们如何共享、转让、公开披露您的个人信息

### 3.1 共享

**默认不共享：** 我们不会向任何第三方出售、出租或以其他方式共享您的个人信息，除非获得您的明确同意或本政策另有规定。

**在以下情况下，我们可能与特定方共享信息：**

**家庭内部共享：**
- 家长用户可以看到其绑定子女的用药记录、健康简报、焦虑量表结果
- 子女用户可以看到其绑定家长的用药提醒设置和部分汇总数据
- 共享仅发生在已建立绑定关系的家庭账户之间

**服务提供商：**
- 微信小程序平台：作为服务载体，微信会按照其隐私政策处理必要的用户信息
- 云服务提供商：用于数据存储和计算（目前为微信云开发）
- 推送服务商：用于发送订阅消息通知

**法律要求：**
- 当法律法规、司法机关或政府机关要求时
- 为维护用户或公众的合法权益所必需时

### 3.2 转让

我们不会将您的个人信息转让给任何公司、组织或个人，但以下情况除外：
- 获得您的明确同意
- 在涉及合并、收购或破产清算时，如发生此类交易，我们将要求新的持有者继续受本隐私政策的约束

### 3.3 公开披露

我们仅在以下情况下公开披露您的个人信息：
- 获得您的明确同意
- 基于法律或司法机关的强制性要求

## 4. 我们如何存储和保护您的个人信息

### 4.1 数据存储

**存储地点：**
- 您的数据存储于中华人民共和国境内的服务器上
- 我们使用微信云开发提供的数据存储服务

**存储期限：**
- 账户数据：保留至您主动注销账户
- 用药记录：保留最近2年的详细数据，更早的数据可能被归档或删除
- 健康简报：保留最近6个月的数据
- 反馈记录：保留最近1年的数据
- 日志信息：保留最长12个月

**数据保留原则：**
- 我们仅在为实现本政策所述目的所必需的期限内保留您的个人信息
- 当法律要求或允许时，我们可能会延长保留期限
- 账户注销后，我们将按照法律规定删除或匿名化处理您的个人信息

### 4.2 安全保护措施

**技术措施：**

**字段级加密（AES-256-CBC）：**
- 对敏感字段（OpenID、昵称、用药记录详情）进行加密存储
- 加密密钥由环境变量管理，定期轮换
- 每个加密值使用随机初始化向量（IV）

**访问控制：**
- 实施严格的数据访问权限管理
- 仅授权人员可访问个人信息
- 所有访问行为被记录和审计

**传输安全：**
- 使用HTTPS/TLS加密传输数据
- 微信小程序与服务器之间的通信经过加密

**组织措施：**
- 建立数据安全管理制度
- 对员工进行隐私保护培训
- 设立数据保护负责人（DPO）联系渠道

**安全事件响应：**
- 建立个人信息安全事件应急预案
- 如发生安全事件，我们将及时通知您并向监管部门报告

## 5. 您的权利

根据适用法律，您享有以下权利：

### 5.1 访问、更正、删除权

- **访问权：** 您有权查看我们持有的关于您的个人信息
- **更正权：** 您有权更新或更正不准确或不完整的个人信息
- **删除权：** 在特定情况下（如目的已实现、您撤回同意等），您有权要求删除您的个人信息

### 5.2 撤回同意权

- 您可以在小程序设置中管理您的授权
- 撤回同意不影响撤回前基于同意已进行的个人信息处理的合法性
- 部分功能（如微信运动同步、订阅消息）需要特定授权，撤回后该功能将无法使用

### 5.3 注销账户权

- 您可以在"设置"页面申请注销账户
- 注销后，我们将停止为您提供服务，并根据适用法律删除或匿名化您的个人信息
- 某些信息可能需要在法定保留期限内继续留存（如交易记录、法律义务）

### 5.4 数据可携带权

- 您有权获取您的个人数据副本，并将其转移给其他服务提供者（如技术上可行）

### 5.5 投诉权

- 如您认为您的个人信息权利受到侵犯，您有权向监管机构投诉
- 中国境内用户可向网信部门投诉
- 欧盟用户可向所在地数据保护机构投诉

### 5.6 如何行使您的权利

您可以通过以下方式行使权利：
- 在小程序内：进入"我的"-"设置"-"账户管理"
- 联系我们：通过小程序内的"意见反馈"功能提交请求
- 我们将在**15个工作日内**响应您的请求（法律另有规定的除外）

## 6. 我们如何处理儿童个人信息

**年龄限制：**
- 本服务面向**18岁及以上**的成年人
- 我们不会故意收集儿童（未满14周岁）的个人信息

**家长监护：**
- 如果您是未成年用户的家长或监护人，并发现我们无意中收集了您子女的信息，请立即联系我们，我们将尽快删除相关数据

**子女功能使用：**
- 子女账户（"子女回声"功能）需要由家长创建和管理
- 子女提交的反馈内容会经过内容安全检查后展示给家长
- 我们不收集子女的精确地理位置等敏感信息

## 附录：法律依据说明（GDPR/PIPL）

根据GDPR第6条和PIPL第13条，我们处理您个人信息的法律依据如下：

| 处理目的 | 法律依据 |
|---------|---------|
| 提供账户服务 | 履行合同所必需 |
| 用药提醒与记录 | 履行合同所必需 |
| 家庭健康数据共享 | 履行合同所必需 + 同意 |
| 微信运动数据同步 | 明确同意（可撤回） |
| 订阅消息推送 | 明确同意（可撤回） |
| 服务改进与分析 | 合法利益 + 匿名化处理 |
| 安全防护与审计 | 履行法律义务 + 合法利益 |

**同意管理：**
- 首次使用时，我们将通过隐私弹窗获取您的明确同意
- 您可以在设置中随时管理您的授权
- 撤回同意不影响此前基于同意已进行的处理
