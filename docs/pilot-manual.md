# 🧑‍🤝‍🧑 内测家庭指南

**欢迎加入「父母周报」内测家庭计划！**

这是一份给非技术家庭的极简使用说明。内测期间我们全程陪伴，有任何疑问随时沟通。

---

## 📋 第一部分：准备工作

### 需要准备什么？

**孩子端（异地图）：**
- 一部能正常使用微信的手机
- 关注本 GitHub 仓库的 Issue（用于提交反馈）
- 每周大概 15-20 分钟，查看简报并提出建议

**父母端（本地）：**
- 一部智能手机（iOS 或 Android）
- 一个能接收短信验证码的手机号
- 每周大概 5 分钟，完成 3-4 个简单任务

### 如何开始？

1. **注册 Demo App**：父母端下载 Demo App（或收到我们发送的安装包）
2. **完成首次登录**：输入手机号验证，完成微信授权关联
3. **试试看**：打开 App，点击「步数打卡」，看看能不能看到反馈

> 📞 遇到问题？在 [Discussions](https://github.com/aitogether/parents-weekly-briefing/discussions) 里告诉我们，我们实时协助。

---

## 📊 第二部分：每周怎么用

### 🗓 父母端（本地）- 每周 5 分钟

**周一到周五（自主行动）：**
- **步数打卡**：每天打开 App，点一下「上报步数」（不需要手动输入）
- **用药确认**：如果当天有服药，点「已服药」；如果漏服，点「忘记啦」
- **情绪自查**：如果有空，花 1 分钟做个小调查（「今天心情怎样？有哪里不舒服？」）

**每周日晚上（关键反馈）：**
- 打开「周报生成」页面，浏览本周摘要
- 点击「发送给子女」——我们会自动推送一封简报给你在异地的孩子

> 💡 提示：父母端不需要输入文字！所有操作都是「点一下」完成的。

### 📬 子女端（异地图）- 每周 15 分钟

**周二或周三（收到简报）：**
- 你会收到一封 GitHub Issue 格式的「父母周报」（或邮件）
- 内容包含：上周步数概览、用药完成率、情绪波动、一句话建议

**周日晚上（轻松互动）：**
- 给父母点个「回声」——一句简单的问候（App 里预设了几个模板）
- 告诉爸妈：「看到你们这周状态不错，我挺放心的」

> 📌 内测期间，我们可能会额外邀请你们填写一个 2 分钟的反馈问卷。

---

## 💬 第三部分：反馈方式（帮助我们把产品做得更好）

内测期间，我们希望每周能从每个家庭收集 5 条关键反馈：

1. **有没有遇到操作卡顿或闪退？** 如果在哪一步卡住了，请截图告诉我们
2. **父母端哪一步操作不直观？** 是没看懂按钮、还是点了没反应？
3. **数据准吗？** 步数、用药记录跟实际是否一致？有没有误差？
4. **简报有用吗？** 子女看这份简报，真的能了解父母状态吗？
5. **还想加什么？** 希望增加什么功能（比如服药提醒、异常预警等）？

### 如何提交反馈？

**首选方式**：在对应 GitHub Issue 下回复（用手机或电脑都可以）
**备用方式**：扫描本仓库 README 里的二维码，加入微信群（限 20 个家庭）

> 🙏 每条真实反馈对我们都极其宝贵。你们的意见会直接影响产品迭代方向。

---

## 🏁 内测期间时间线

- **第 1-2 周**：安装验证 + 基本功能跑通
- **第 3-4 周**：跑通第一轮完整周报流程
- **第 5-8 周**：收集反馈、调整交互、优化体验

内测结束后，我们会给每个家庭一份「内测纪念证书」+ 正式版终身免费使用权 🎁
