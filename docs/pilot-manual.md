# 内测使用说明 · Parents Weekly Briefing

> 版本：P0 测试版 | 更新日期：2026-03-22

---

## 子女端

### 如何使用
1. 在微信中搜索小程序（或扫码导入体验版）
2. 首次进入后绑定父母信息（姓名、角色）
3. 开启微信运动授权

### 你能看到什么
- **每周一 9:00**：收到「父母周报」，包含本周总评、爸妈各自的步数/用药状态
- **每天中午 12:00**：收到「每日小结」，查看当天用药和活动状态
- **每周五 17:00**：收到「周五回访」，提醒你这周有没有跟爸妈聊过

### 操作步骤
1. **查看周报**：打开小程序 → 周报页（绿灯=不用操心，黄灯=留意，红灯=需关注）
2. **查看用药**：小程序 → 用药统计页 → 查看完成率
3. **添加用药计划**：小程序 → 设置页 → 添加药品名称、剂量、服药时间
4. **回声**：周报底部选择三句话之一（「一切放心」/「有点担心」/「一直惦记」），父母用药页会看到你的话

## 父母端

### 如何使用
1. 用父母的微信扫码导入小程序
2. 界面为**大字体、大按钮**设计
3. 不需要复杂操作，只有两个核心功能

### 核心功能
- **用药提醒**：每天到服药时间会弹窗提醒
  - 点**绿色大按钮「已吃」** → 完成确认
  - 点**红色大按钮「没吃」** → 记录跳过
- **查看用药计划**：可以看到每天该吃什么药

### 注意事项
- 父母端只有用药确认功能，不需要看周报
- 确认按钮**只有父母角色能点击**（子女端看到的是只读汇总）
