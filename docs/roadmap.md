# 产品路线图 · Roadmap

## 已完成 ✅

- [x] 周报详情页原型（HTML 高保真 + Android Demo）
- [x] 微信小程序 MVP（周报 / 用药确认 / 回声 4 个页面）

## 2026 Q1（1–3 月）— 内测验证

- [ ] **云开发 PoC**：小程序接入微信云开发，验证免部署后端方案可行性
- [ ] **5–10 个家庭内测**：收集真实使用反馈，重点关注：
  - 父母端大按钮交互是否足够简单
  - 周报信息密度是否合适
  - 「回声」功能的使用意愿
- [ ] 内测反馈整理与优先级排序

## 2026 Q2（4–6 月）— 上架准备

- [ ] 根据内测反馈迭代 UI 和交互
- [ ] 小程序提交审核 & 上架
- [ ] 建立公测用户群，扩大至 30–50 个家庭
- [ ] 后端性能优化（如采用云开发则同步迁移）

## 2026 Q3（7–9 月）— 公测反馈收集

- [ ] 公测上线，开放注册
- [ ] 收集公测数据：留存率、周报打开率、回声发送率
- [ ] 评估商业模式：免费增值 vs 订阅制 vs 按次付费
- [ ] 探索商业化合作（社区养老、保险公司等）
