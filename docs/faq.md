# ❓ 常见问题（FAQ）

> 本文档针对 P0 内测用户。正式版发布后可能有所调整。

---

## 🎯 内测相关

### Q：内测持续多久？如何退出？
**A**：内测为期 2 周（Week 1 + Week 2）。退出方式：
1. 在小程序「我的」→「退出内测」中自助退出
2. 或联系开发者手动删除数据
退出后所有数据在 48 小时内清除，之后可正常使用或卸载。

### Q：内测期间收费吗？
**A**：完全免费。内测结束后，如果您选择继续使用，基础功能（周报+用药确认+回声）将永久免费（CC BY-NC 4.0 许可证）。未来可能推出付费增值功能，但不会影响现有免费功能。

### Q：一个家庭可以绑定多个子女吗？
**A**：当前版本 **不支持**。一位父母只能绑定 1 位子女，一位子女也只能关注 1 对父母。P1 版本计划支持多对多绑定。

### Q：内测名额有限，可以邀请其他家庭吗？
**A**：可以！内测用户享有「邀请资格」，可额外邀请 1 个家庭共同测试（双方均获额外权益，如延长免费期）。请联系开发者获取专属邀请码。

## 📱 小程序使用

### Q：父母端收不到用药提醒推送怎么办？
**A**：请按顺序检查：
1. 微信通知权限是否开启（手机设置 → 微信 → 通知 → 允许通知）
2. 勿扰模式是否关闭（控制中心月亮图标）
3. 小程序是否被"最小化"（从后台彻底关闭再重启）
4. 网络是否正常（尝试打开网页）

如果仍无效，请提交 GitHub Issue 并附上：
- 手机型号（如 iPhone SE 2020）
- 微信版本（我 → 设置 → 关于微信）
- 操作系统版本（iOS 15.4 / Android 12）

### Q：父母忘记输入邀请码了怎么办？
**A**：邀请码有效期 48 小时，过期后请让子女在「添加父母」页面重新生成。新码生成后，旧码自动失效。

### Q：子女端看不到父母的周报数据？
**A**：请确认：
1. 父母是否已绑定成功（子女端首页应显示父母姓名）
2. 父母是否已完成本周至少 1 次用药确认
3. 时间是否为周五晚上 8 点后（周报定时生成）
4. 网络是否正常（尝试下拉刷新）

如果以上都正常但仍无数据，请检查后端服务是否在线，或提交 Issue。

### Q：步数数据不更新或显示为 0？
**A**：可能原因：
1. 父母当天未携带手机（步数依赖手机传感器）
2. 微信运动未开启（微信 → 我 → 设置 → 通用 → 辅助功能 → 微信运动 → 启用）
3. 数据有 1-2 小时延迟，请在晚上 8 点后查看
4. 如果连续 3 天为 0，请检查微信运动授权是否正常

### Q：可以补确认昨天的用药吗？
**A**：可以。父母端进入用药页面后，可手动选择昨天的日期进行补确认。补确认会计入本周统计。

### Q：子女发送的"回声"消息，父母能看到多久？
**A**：消息会永久保留在父母端首页顶部，直到被新的"回声"覆盖（每周一条）。如果您希望某条消息长期保留，可以建议我们增加"收藏"功能。

## 🔐 隐私与安全

### Q：你们会收集我的位置信息吗？
**A**：不会。我们只收集微信运动的**步数汇总值**（每天总计），不收集 GPS 轨迹、不记录您去过哪里。

### Q：子女能看到我每天的具体步数吗？
**A**：子女只能看到**周度折线图**（7 条数据点），无法看到每小时步数。如果您某天步数为 0，子女会看到"该天无数据"，但不会知道您全天没出门。

### Q：如果子女和父母分开了（如离婚），可以解绑吗？
**A**：可以。双方都可以在小程序内点击「解除绑定」，解绑后数据将被清除，需重新绑定。

### Q：我的数据会泄露给第三方吗？
**A**：不会。我们不出售数据，不与任何第三方共享数据（法律要求除外）。详细条款见 `docs/privacy-and-security.md`。

### Q：开发者能直接看到我的数据吗？
**A**：能，但仅限于调试 bug 和生成周报。开发者只有 1 人（独立开发者），不会滥用数据。您也可以选择退出内测，要求彻底删除所有数据。

## 🛠️ 技术问题

### Q：支持华为鸿蒙系统吗？
**A**：支持。微信小程序在 HarmonyOS 上通过系统内置微信运行，功能与 iOS/Android 无异。但注意：鸿蒙版微信可能暂时未集成微信运动功能，届时步数数据可能无法同步。

### Q：可以同时关注多对父母吗（如外公外婆+自己父母）？
**A**：当前版本 **不支持**。一位子女只能绑定 1 对父母。P1 版本计划支持"多父母关注"。

### Q：如果父母不用智能手机怎么办？
**A**：当前版本**必须**有智能手机（Android/iOS）并安装微信。如果父母完全不用智能手机，本工具暂不适用。未来可考虑简化版（短信确认用药），但不在当前路线图。

### Q：用药提醒可以修改时间吗？
**A**：当前提醒时间为**每天中午 12:00**（固定），暂不支持自定义。P1 版本计划支持个性化提醒时间设置。

### Q：可以添加多个用药计划吗（如早、中、晚）？
**A**：可以。子女端可为父母添加多个用药计划（如早餐后、晚餐后），父母端会一次性列出当天所有计划，只需点一次"已吃药"即代表所有药都已服用（简化设计）。

### Q：周报可以手动触发生成吗？
**A**：不可以。周报**每周五晚 8:00 自动生成**，这是为了培养使用习惯，避免子女过度查看（焦虑）。P1 版本可能增加"随时查看本周数据"功能。

## 💰 商业与未来

### Q：未来会收费吗？收费模式是什么？
**A**：基础功能（周报+用药确认+回声）**永久免费**（CC BY-NC 4.0）。未来可能推出付费增值功能，如：
- 更多历史数据回看（3个月/1年）
- 多父母绑定
- 语音消息
- 定制化 AI 建议

所有付费功能**不影响**现有免费功能，且内测用户享有永久免费使用权。

### Q：这个项目会一直维护下去吗？
**A**：作为独立开发者，我会尽力维护。如果未来因故停止服务，会提前 3 个月在 GitHub 和微信小程序内公告，并开放数据导出功能。

### Q：可以与医院系统对接吗（如电子病历）？
**A**：短期不会。涉及医疗数据合规性（HIPAA/GDPR/中国个人信息保护法），成本极高。本工具定位为**非医疗工具**，不提供诊断建议，仅提供"关怀提醒"。
